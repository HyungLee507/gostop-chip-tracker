# gostop-chip-tracker

고스톱 칩 이동을 기록하고 정산까지 계산해주는 모바일 웹앱입니다.
3~4명이 동일한 시작 칩으로 게임을 시작하고, 매 판 끝날 때마다 플레이어 간 칩 이동만 기록하면 됩니다.

## 기능

### 게임 설정
- 3~4명 플레이어 이름 설정 (기본값 제공)
- 동일 시작 칩 수 설정
- 게임 초기화

### 칩 이동 기록
- 플레이어 카드를 탭하여 보내는/받는 사람 선택
- 숫자 패드로 이동량 입력
- 실시간 잔액 및 순위 표시
- 시작 칩 대비 증감 표시

### 거래 내역
- 전체 칩 이동 내역 확인 (최신순)
- 개별 거래 삭제 가능

### 정산 계산
- 최종 순위 표시
- 칩당 금액 설정 (원 단위)
- 최소 정산 경로 자동 계산 (누가 누구에게 얼마를 줘야 하는지)

## 기술 스택
- React 18 + Vite
- localStorage (백엔드 없음)
- 모바일 반응형 CSS

## 실행
```bash
npm install
npm run dev      # 개발 서버 (localhost:5173)
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 미리보기
```

## 프로젝트 구조
```
src/
├── main.jsx      # 엔트리 포인트
├── App.jsx       # 메인 컴포넌트 (게임 로직 전체)
└── styles.css    # 스타일
```
